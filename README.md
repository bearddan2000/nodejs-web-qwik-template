# nodejs-web-qwik-template

## Description
A template for the qwik framework using an express server.

## Tech stack
- npm
- express
- qwik
- vite

## Docker stack
- node

## To run
`sudo ./install.sh -u`
- [Availble here](http://localhost)

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
